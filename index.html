Jan Moesen's bookmarklets

Time-saving and life-enhancing (ahem) bookmarklets.

Installation instructions

Download bookmarks.html and import it into your bookmarks manager.

Browser compatibility

I use Firefox for work and leisure, and all bookmarklets should work in the latest stable release.

Note that a lot of my bookmarklets use E4X. Because E4X is not widely supported, and might even disappear completely, the bookmarklets that accept parameters (e.g. len This "len" bookmarklet shows a string's length.) will only work in Mozilla browsers like Firefox or SeaMonkey.

The bookmarklets I use the most (Readable++, « Previous and Next ») also work fine in other browsers I use, like Google Chrome and Safari on iOS.

Source code

You can clone the project with Git by running: 

$ git clone git://github.com/janmoesen/bookmarklets
